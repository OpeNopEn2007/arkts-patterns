# ArkTS 开源项目资源

> 持续更新中...

## 精选学习项目

### 1. HarmonyNEXT_Examples ⭐ 推荐首选
- **URL**: https://github.com/RicardoWesleyli/HarmonyNEXT_Examples
- **特点**: 全流程 HarmonyOS NEXT 原生开发学习项目
- **技术栈**: ArkTS + ArkUI
- **学习价值**:
  - 完整的 ArkUI 声明式开发范式示例
  - 覆盖 HarmonyOS NEXT 开发全流程
  - 适合系统学习 ArkTS 语言特性

### 2. awesome-harmony ⭐ 资源汇总
- **URL**: https://github.com/Wscats/awesome-harmony
- **Stars**: 17.7K+
- **特点**: 鸿蒙系统开发资料精选集合
- **内容**: OpenHarmony JS Demo、组件教程、开发流程

### 3. HarmonyOS/Samples (官方示例)
- **URL**: https://gitee.com/OpenHarmony-SIG/HarmonyOS_Samples
- **Stars**: 688+
- **特点**: 官方 SDK 示例集合
- **内容**: 媒体、图形、AI 等六大领域能力

### 4. harmony-utils (工具库)
- **URL**: https://gitee.com/lian-xian-sen/harmony-utils
- **Stars**: 536+
- **特点**: 功能丰富的 HarmonyOS 工具库
- **功能**: APP、设备、屏幕、授权、通知、加密、日期处理等

### 5. HarmonyComponent (组件示例)
- **URL**: https://gitee.com/Minzhang/HarmonyComponent
- **特点**: 鸿蒙应用程序组件示例集合
- **学习价值**: 可复用的 UI 组件设计模式

---

## 学习路径建议

1. **入门阶段**: HarmonyNEXT_Examples - 了解 ArkTS 基础语法
2. **进阶学习**: awesome-harmony - 深入学习汇总资料
3. **组件设计**: HarmonyComponent - 学习组件封装模式
4. **工具封装**: harmony-utils - 学习工具类最佳实践
5. **官方示例**: SDK samples - 掌握系统 API
